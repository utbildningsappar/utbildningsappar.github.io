Privacy Policy

This document explains what data is collected from our apps, Körkort Moped Gratis and Körkort Moped Plus, and how this data is used

Our apps, Körkort Moped Gratis and Körkort Moped Plus, include forms where you can send feedback and report issues. We use the data you send to us in these forms to service your requests and fix reported issues. We do not share the data you send to us in these forms with anyone. Once a request has been handled, the data is promptly deleted.

In our apps, we use Firebase, which is a software library provided by Alphabet Inc (Google). Firebase collects a bundle of information called the Firebase user agent. The Firebase user agent includes information on device type, OS, app bundle ID, and developer platform. The Firebase user agent is not linked to your identity. The firebase user agent is used by the Firebase team to better understand how Firebase is used by developers. You can read more about the Firebase user agent at https://firebase.google.com/docs/ios/app-store-data-collection.

To see and request removal of your data, please contact us at info@utbildningsappar.com.
